08-Relative Units


Specify a length relative to another length property.

  • Pixel (px)
  • M (em)
  • X (ex)
  • Root M (rem)
  • Viewports units horizontal (vh)
  • Viewports units width (vw)
  • Large viewport height (lvh|lvw)
  • Small viewport height (svh|svw)
  • Dinamic viewport (dvh|dvw)
  • px - 15px
  • em - 1.5em
  • rem - 2rem
  • ex - 2.4ex
