Work Flow - Stylling

Nesting >

Example

Result

Sibling >

Example

section+header+nav ↔️

Result

Repeat >

Example

ul>li*3 ↔️

Result

Class >

Example

<div class="contenedor"></div>


Multiple Classes >

Example

div.clase1.clase2.clasen ↔️

Result

<div class="clase1 clase2 clasen"></div>
